ปั้น Workflow ด้วย AI ใน n8n ยังไงให้เวิร์ค?
ตอนนี้ n8n มีตัวช่วยสร้าง Workflow ด้วย AI (AI Workflow Builder) แล้วนะ แค่พิมพ์บอกมันก็สร้างออกมาเป็น Node ให้เลย แต่ถ้าอยากให้มันทำออกมาได้เป๊ะที่สุด นี่คือเทคนิคที่ n8n เขาแนะนำมาครับ

เทคนิคเด็ดๆ ที่ควรรู้:

ค่อยๆ ปั้นทีละสเต็ป: อย่าเพิ่งพิมพ์สั่งงานยาวเหยียดในทีเดียว ให้เริ่มจากโครงร่างคร่าวๆ ก่อน แล้วค่อยๆ สั่งให้ AI ปรับแก้หรือเพิ่มรายละเอียดเข้าไปทีละนิด (Iterative building) จะได้ผลลัพธ์ที่แม่นยำกว่าครับ

เจาะจงชื่อเครื่องมือไปเลย: แทนที่จะบอกว่า "ส่งอีเมล" ให้บอกไปเลยว่า "ส่งอีเมลผ่าน Gmail" หรือแทนที่จะบอกว่า "ลงตาราง" ให้ระบุว่า "เพิ่มข้อมูลลงใน Google Sheets" AI จะเลือก Node ได้ถูกต้องทันที

อธิบายการไหลของข้อมูลให้ชัด: บอกมันไปเลยว่าให้ไปเอาข้อมูลจากไหน ทำอะไรต่อ และสุดท้ายไปจบที่ไหน เช่น "ดึงเมลล่าสุด 10 ฉบับจาก Gmail -> ให้ OpenAI สรุปเป็นข้อๆ -> แล้วเอาไปลงใน Google Sheets ชื่อ Report"

ไม่ต้องสวมบทบาท (Roleplay): ไม่ต้องไปพิมพ์ว่า "คุณคือผู้เชี่ยวชาญ n8n" นะครับ เสียเวลาเปล่าๆ เพราะตัวมันถูกเซ็ตมาให้เก่งเรื่องนี้อยู่แล้ว สั่งงานตรงประเด็นไปได้เลย

เตรียม Credentials ให้พร้อม: AI สร้าง Node ให้ได้ แต่มันล็อกอินให้เราไม่ได้นะ พอสร้างเสร็จอย่าลืมไปใส่ API Key หรือเชื่อมต่อบัญชีต่างๆ ให้เรียบร้อยด้วย

อย่าพึ่ง AI ตัวอื่นเจน Prompt ให้: บางทีการไปก๊อป Prompt ยาวๆ จาก Chatbot ตัวอื่นมาวาง อาจจะทำให้ AI งงได้ พิมพ์สั่งสั้นๆ กระชับ และตรงไปตรงมาใน n8n เองจะได้ผลดีที่สุดครับ

สรุปคือ ยิ่งเราชัดเจนและแบ่งงานเป็นส่วนย่อยๆ AI ก็ยิ่งช่วยเราประหยัดเวลาสร้างระบบ Automation ได้มหาศาลเลยล่ะ!
