# Agentic Blogger

Agentic Blogger is an AI-powered platform that generates content for bloggers.

## Features

1. LLM read rss feed from many sources we want.
2. LLM generate content based on the rss feed.
3. LLM generate meta description and title.
4. LLM use unsplash API to seach and hotlink image for the content.
5. LLM generate markdown file for the content.
6. Approval process for the content via discord chat.
7. Auto publish the content to the blog via github API.
