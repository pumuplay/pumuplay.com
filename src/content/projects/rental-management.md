# Rental Management

Rental management system without software development.

## Features

- Owner manage rentals from notion database
- Renter can book a rental from chatbot
- Renter can pay for the rental from chatbot
- Renter can cancel the rental from chatbot
- Renter can ask questions to the owner from chatbot
- Owner can manage rentals from chatbot
- Owner can send bill to the renter from chatbot
- Access Chatbot via LINE OA
