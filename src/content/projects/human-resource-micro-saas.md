# Human Resource Micro SaaS

Human Resource Micro SaaS is a micro SaaS that helps HR teams to manage their employee attendance.

## Tech Stack

- Bun bundler
- Vite
- ReactJS
- Tailwind CSS
- Supabase
- Prisma
- TypeScript
